end_roll()[source]¶

Called when player has finished movement of piece. Resets all player’s pieces.movable to FALSE. Checks if player should end turn, Otherwise allows player another dice roll. Checks if all player’s pieces on home run, allowing player to win. :var self.my_player.roll: Value of dice :var self.my_player.special_move: Boolean used when player piece kill opposing piece :var self.my_player.rolls_taken: Counts amounts of times player has rolled dice :var self.my_player.diceroll_token: Boolean used to prevent player rolling dice in succession

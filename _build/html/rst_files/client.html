client module¶

class client.Ludo[source]¶

Bases: object

This is the main Ludo class.

It initialises my_player, genie_owner, all_pieces, board, connection, the score board and the timer.

It also holds the main run function which runs the game.

Initialises the main attributes but does not take in any arguments.

bgm()[source]¶

Sets the frequency, loads the background music and plays it.

click_piece(num, piece)[source]¶
Parameters:num – a number

After a dice is roller, if the player clicks a movable piece, call click_piece. It calls the move_piece function, it also sends what piece was moved to the server.

draw_Time_Out()[source]¶

Draws the timer which counts down until it reaches 0. When this happens it goes back to its original number and counts down again.

run()[source]¶

This is the main game method. It draws the board, pieces and the buttons. It also shows the diceS rolling animation.

setup()[source]¶

Creates the coo-rdinate dictionary for the board, initialises pygame. It also blocks out some pygame events so the queue doesn’t overflow from unneeded events such as MOUSEMOTION. It also setups attributes in board, connects to the server and shows the start screen.

show_start_screen()[source]¶

Shows the start screen whent game first starts.

It shows the word LUDO in flashing colours. Once the player connects to the server, the screen goes away.

terminate()[source]¶

Quit game if user closes window.
